# py_base
Projeto base para python

## Instalação
- criar *virtualenv* para o python na raíz do projeto:
```
python -m venv ./venv
```

- ativar o ambiente virtual do python:
```
source ./venv/bin/activate
```

- instalar os pacotes:
```
pip install -r requirements.txt
```

- criar os hooks do *pre-commit*:
```
pre-commit install
```
